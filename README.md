
Selenium Automation Project (Assignment 15 – Module 23)
Project Overview

This project is part of Assignment 15 from Module 23: Automation using Selenium – Getting Data. The goal of this script is to automate a browser using Selenium, perform a search operation, and extract titles from the search results.

***Project Features***

Opens Google automatically
Searches for a given query ("Selenium Python tutorial")
Extracts all result titles from the page
Prints extracted data in the console
Closes the browser after execution
Technologies Used
Python 3
Selenium WebDriver
ChromeDriver

***How to Run the Project***
1. Install Dependencies

Run this command:

pip install selenium
2. Download ChromeDriver

Make sure the ChromeDriver version matches your Chrome browser. Download from: https://chromedriver.chromium.org/

Place the driver in the project folder or set it in PATH.

3. Run the Script

Execute this command:

python selenium_project.py
